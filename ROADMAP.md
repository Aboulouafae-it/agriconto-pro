# Roadmap

This root roadmap summarizes the product direction. See [docs/ROADMAP.md](docs/ROADMAP.md) for details.

## Phase 1: Core MVP

- Goal: farm-scoped accounting and workforce foundation.
- Status: in progress.

## Phase 2: Desktop Stabilization

- Goal: reliable Debian/Linux desktop app.
- Status: in progress.

## Phase 3: Professional PDF Reports

- Goal: accountant-ready management reports.
- Status: in progress.

## Phase 4: Advanced Analytics

- Goal: deeper business intelligence for farms and advisors.
- Status: foundation implemented, expansion planned.

## Phase 5: Local Backend Integration

- Goal: bundled local backend runtime for desktop users.
- Status: planned.

## Phase 6: Mobile/PWA Field Mode

- Goal: field-friendly mobile workflows.
- Status: planned.

## Phase 7: OCR and AI Assistant

- Goal: document capture and assistant workflows.
- Status: planned.

## Phase 8: Cloud/SaaS and Multi-Farm Accountant Workspace

- Goal: hosted collaboration and accountant workspace.
- Status: planned.

# Release Checklist

Use the detailed checklist in [docs/RELEASE_CHECKLIST.md](docs/RELEASE_CHECKLIST.md).

Minimum release checks:

- Version updated.
- Changelog updated.
- README updated.
- No secrets committed.
- Backend health endpoint works.
- Frontend build passes.
- Desktop package builds.
- PDF export tested.
- Login tested.
- Main pages tested.
- Security-sensitive permissions reviewed.

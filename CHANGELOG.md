# Changelog

This project follows the spirit of [Keep a Changelog](https://keepachangelog.com/) and uses semantic versioning while the MVP stabilizes.

## Unreleased

### Added

## 0.1.0

### Added

- FastAPI backend foundation with PostgreSQL, SQLAlchemy and Alembic.
- JWT authentication and role-based access controls.
- Farm membership isolation using `farm_id` as tenant boundary.
- Italian-first React/Vite frontend.
- Core dashboard.
- Workers, workdays, fields, crops, expenses, sales and documents pages.
- Commercialista page and Pacchetto Commercialista workflow.
- Analytics foundation for Statistiche e Analisi.
- JSON report services and PDF export foundation.
- Secure document upload/download foundation.
- Audit logging foundation.
- Electron desktop wrapper.
- Debian/Linux packaging foundation with `.deb` and AppImage.

# Product

AgriConto Pro organizes farm financial, labor and document data for Italian agricultural businesses.

It helps owners prepare clean records for accountants and labor consultants. It does not file taxes, replace advisors, calculate official payroll, or submit official communications.

Core modules:

- Aziende agricole and members
- Lavoratori
- Campi and colture
- Giornate di lavoro
- Anticipi and pagamenti lavoratori
- Spese and vendite
- Fornitori and clienti
- Documenti and richieste documentali
- Report JSON for accountant-ready summaries
